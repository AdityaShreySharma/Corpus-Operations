## Requirements
1) nltk
``` py
import nltk
nltk.download()
```

2) Ghostscript

Download from the link given below.
```
https://www.ghostscript.com/releases/gsdnld.html
```
Add the Path (For Example - C:\Program Files\gs\gs9.55.0\bin) to the Environment Variables.
